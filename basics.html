Programming Languages


It can be overwhelming to pick where to start, so below is a short summary of the most popular programming languages to help you get started. Not every programming language is listed here, just the most popular.


HTML/CSS


The foundational languages of every webpage, including this one. If you learn HTML, you will have the power to structure and build your own webpage, and with CSS, you will be able to make it look amazing. Choose this if you are interested in Web Design.



Python


Python is likely the most popular programming language out there, and is very good for beginners because of its easy to read syntax and beginner friendly format. It also can do basically anything, from web development to data science. Choose this if you are interested in solving problems and competing in programming.



Java


Java (no, not coffee, sorry) is also widely popular, and mainly used for developing large systems and software. It is slightly less streamlined than Python, but still good for beginners. Choose this language if you are interested in competing and writinig powerful programs.



Javascript


Javascript is fun and flexible, and also often a core language used to build webpages. Choose this if you want a smooth introduction to programming while still learning Web Development.



C#


C# is Microsoft's language of choice, and is used to build games, apps, websites, and pretty much everything under the sun. Choose this if you want something more off the beaten path.



C++


C++ is a general purpose language that is similar to C#. Ideal for those who aren't interested in Python.
