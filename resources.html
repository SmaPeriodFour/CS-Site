Helpful Resources


The links below will help you get started with code. If you're unsure where to start, take our QUIZ.




Codecademy


Offers free and paid tutorials for most all programming languages. Codecademy is an excellent beginner and intermediate level resource and even includes walkthroughs on projects, which can help you build your programming portfolio.


W3Schools


One of the most popular code tutorial sites out there. Offers free tutorials and paid certification courses for most major languages. Features practice exercises at the end of every lesson.


CodeWars


An intermediate-level coding resource designed to aid users in solving coding problems through the use of efficient algorithms.


LeetCode


An intermediate to advanced-level resource that includes ONLY practice problems for technical interviews (the types of problems in CS competitions).


USACO Training Gateway


An intermediate to advanced-level coding resource that is very similar to LeetCode. This will most likely be the primary resource we will be using to prepare for competitions.


coursera


An advanced-level resource containing courses from a multitude of prestigious universities, including Stanford University, Yale and Princeton to name a few. You can earn certification for completing these courses, and these courses look great on college applications!
